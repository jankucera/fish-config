fish-config
===========

Configuration files for fish shell - [fishshell.com](http://fishshell.com).


## Install fish 
```
brew install fish
```

on M1 chip:
```
arch -x86_64 brew install fish
```


## Config

Download git repository
```
git clone https://github.com/jankucera/fish-config.git ~/.config/fish/
```
